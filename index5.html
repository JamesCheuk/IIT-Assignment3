Scope & Limits

Scope

The project's scope will be restricted to developing a simple app for an Android or iOS device that will allow users to select a destination they would like an alarm to be triggered as they approach or arrive at it.

The App will consist of 2 tiles a user will need to navigate through to set the alarm. The following capability will be available on each of the tiles:

Tile 1

Current Location

Using the device's default map application automatically displays the current location based on GPS coordinates captured

End Destination

Allow users to search and select an address or location they would like an alarm triggered by their mobile device.

Tile 2

Select Distance from Destination Alarm

Allow a user to select the distance from the destination the alarm is to be triggered. This distance may be between 1 kilometre to 50 metres from the end destination.

Select Sound for Alarm

Allows a user to select the alert's sound, which is triggered as they arrive at their destination. The selection of different sounds for the alarm will be restricted to five different sounds inbuilt into the App.

Confirm your Alarm

Allows the user to initiate the alert so that their mobile device can track their location via GPS to enable the alarm to be triggered once they approach or arrive at their destination.

Distribution Channels

Wake App will only be distributed through the Android Play Store or Apple’s App Store.

Limits

As part of the Wake App's initial release, only the App's necessary foundations will be developed. This includes excluding the following additional capabilities associated with the development of the App:

  • Personalising the sound of the alert
  • Enabling the user search and select different start location of the alarm
  • Enabling the user to select travel mode to the destination such as a car, public transport, bike, and/or walking.
